What is Machine Learning¶

Machine learning is a subfield of artificial intelligence. Its goal is to enable computers to learn on their own. A machines learning algorithm enables it to identify patterns in observed data, build models that explain the world, and predict things without having explicit pre-programmed rules and models.

Applications of Machine Learning¶

Machine learning has applications in many fields. A (non-exaustive) list of applications includes spam detection, generating search results, image classification, voice recognition, programming autonomous vehicles, predicting credit scores, predicting housing costs, medical diagnoses, recommender systems, and handwriting recognition.

Model Building¶

The goal of machine learning is to construct models that allow us to identify trends and make predictions. A machine learning algorithm "trains" on data that is provided to it, and then outputs a model that can be used to generalize beyond the training data.

Types of Machine Learning Algorithms¶

There are three main categories of machine learning problems: supervised learning, unsupervised learning, and reinforcement learning.

Supervised Learning¶

In a supervised learning task, the goal is to generate a model that can predict an output based on one or many input values.

  • The output is typically called a label, target value, or response variable.
  • The inputs are typically called features or predictors.

The algorithm "learns" the optimal model by studying a training set of data that consists of several observations (or instances, or samples), each of which contains values for both its features and its label.

Supervised learning tasks can be further classified into regression and classification problems.

  • Regression. A regression task is one for which the target values are continuous, real numbers.
  • Classification. A classification task is one for which the target values are discrete classes.

Objective Functions¶

A supervised learning algorithm uses the data it is provided to learn the optimal model for the data it was provided. It determines which model is optimal by trying to either minimize or maximize an objective function that is used to score the model.

Training a Regression Algorithm: Sum of Squared Errors¶

For regression tasks, a common objective function is the sum of squared errors, or SSE. Assume that our training set has $n$ observations, whose labels are denoted by $y_1, y_2, ..., y_n$. Assume that we create a regression model, and based on the features of the observations in our training set, the model predicts outputs of $\hat{y}_1, \hat{y}_2, ..., \hat{y}_n$. For each observation, we can define the error of the prediction to be $\hat{e}_i = \hat{y}_i - y_i$. Having calculated the errors for each of our training instances, we can then calculate the SSE as follows:

$\hspace{10 mm} \large SSE = \sum \hat{e}_i^2 $

The goal of the algorithm is the find the model that produces the lowest SSE. Consider the following example, which illustrates the linear regression algorithm.

Training a Classification Algorithm: Accuracy¶

A common objective function used for scoring a classification algorithm is accuracy. After using the training data to generate a classification model, we can then apply that model to each of the instances in the training data to generate class predictions for each instance. The accuracy of the model is simply the proportion of instances that have been correctly classified. The goal of the learning algorithm is to find the algorithm with the highest accuracy.

Unsupervised Learning¶

In unsupervised learning, we work with data sets in which features are provided, but no labels. Not only are the labels "missing", but we typically do not start an unsupervised learning tasks with a preconception of what the possible labels might be. The goal of unsupervised learning is to identify structure within the data that might not be readily perceptible to humans.

Two common unsupervised learning tasks are:

  • Clustering
  • Outlier Detection

Reinforcement Learning¶

In reinforcement learning task, a notion of "success" is defined, and a method of score the level of success is provided. The algoirthm then collects a training set through trial-and-error. Which each new attempt, the algorithm recieves feedback regarding its performance, and then makes adjustments as needed.
